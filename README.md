# 🔐 DPDP Privacy Experience Platform

> **A Unified Privacy & Consent Framework Across India's Leading Consumer Applications**

An interactive **Product Management portfolio** showcasing how India's **Digital Personal Data Protection (DPDP) Act, 2023** can be implemented across multiple consumer products through privacy-first experiences, consent management, user rights, analytics, and AI-assisted guidance.

---

## Overview

The **DPDP Privacy Experience Platform** is a unified Product Management case study that integrates **five interactive privacy-first prototypes** into a single experience while preserving each product's original design, interactions, and functionality.

Instead of redesigning each product separately, this platform demonstrates how a reusable privacy framework can be adapted across different consumer applications.

---

## 🚀 Products Included

- 🎓 Vedantu
- 🚗 Uber
- 🎵 Spotify
- 📺 YouTube
- 🍽️ Zomato

Each prototype is embedded and preserved exactly as its original standalone implementation.
